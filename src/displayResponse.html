Displaying analysis results

Now that we have successfully requested image analysis, we need to display the response data returned from the API.
We've added quite a bit to our program where we requested image analysis, but the additions are mostly just listings of the image analysis response. We set up a string of code that includes the key name, a colon, and then the corresponding value for the pair. This is then set equal to the textContent of our new variable and appended to a newly created unordered list element. While we used a list format in this example, your element could have been a table cell, etc.

After creating a list element, we then add a returned value pair to that list element. Our response data was returned as an array of objects, nested in an object, so the processing is a little lengthy.
